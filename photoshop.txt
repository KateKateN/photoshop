Інтерфейс

	1. Вертикально - панель інструментів, кожна з яких - група.
		(Правий клік по іконці покаже групу)
	2. Горизонтально - налаштування активного інструменту.
	3. Правий бік - додаткові панелі.
	4. Показати і сховати будь-яку панель - меню Window.
	5. Верстальщикам потрібно тільки 2 панелі:
		Layers (слої) F7
		Character (символи - відомості про шрифти)
	6. Покажемо їх і винесемо у робочу область.
	7. Збереження наших налаштувань:
		верхній правий бік - 'Essentials' - 'New Workspace' - Name: web - Save

Налаштування

	1. Mеню Edit - Color Settings... Ctrl+Shift+K (має бути для RGB вибрано sRGB).
	2. Mеню Edit - Preferences - Interface... Color Theme: (правий крайній квадрат - дає невисокий контраст і меньше подразнює очі) - ОК.
	3. Mеню Edit - Preferences - Units & Rules... (лінійки та шрифти) всім задаємо 'рх' - ОК.

Орієнтація по макету і горячі клавіші

	1. Відкриваємо будь-який макет.
	2. Ctrl+1 - масштабування до 100%.
	3. Ctrl+0 - показати цілий макет.
	4. Зажатий Alt+колесо миші - масштабування того місця, де знаходиться курсор.
	5. Зажатий Пробіл+Ліва кнопка миші - лупа - пересування макету.
	6. F12 - відновлення відкритого документу до стану останнього збереження на диск.
	7. Ctrl+Shift+Alt+S - Save for WEB.
	8. Ctrl+N - створення нового документу.
	9. Ctrl+S - зберігання.
	10. Ctrl+W - закрити.

Показ і приховування допоміжних елементів (меню View - Show)

	1. Ctrl+Н - чередування показу і приховування направляючих ліній.
	2. Ctrl+R - чередування показу і приховування лінійки.

Орієнтація по слоях, модель слоїв

	1. Робота з Фотошопом - це робота з слоями (панель Layers).
	2. Є фільтр: по типу, імені... Його можна вмикати і вимикати.
	3. Якщо відфільтрувати слої по типу 'текст' - можна побачити всі шрифти, які відсутні в операційній системі (жовтий трикутник).
	4. Режими в макетах сайтів мають бути 'Normal' або 'Pass Throught', якщо інші - 'Недодизайнер', звернення за 'нормальним макетом'.
	5. Слої можуть бути блоковані повністю, або можуть бути блоковані зміни властивостей слоя, наприклад, можливість переміщення.
	6. Іконка ока - показ і приховування слоя.
	7. Зажатий Alt+іконка ока - залишиться видимий тільки слой, поруч з яким клікнули. Ще один клік - попередній стан.
	8. Якщо іконка ока не стоїть навпроти якогось ефекта слою - ефект не видно та верстати його не потрібно.
	9. Робота з слоями - меню Layer та низ панелі Layers.
	10. Маски.
	11. Подвійний клік по назві ефекту відкриває нам сам ефект, активні ефекти відповідно відмічені.
	12. Подвійний клік по мініатюрі кольору дасть нам колір залівки.

Ручний спосіб отримання даних про шрифт

	1. Обираємо інструмент роботи з шріфтом.
	2. Клікаємо та обираємо той фрагмент, який нас цікавить.
	3. Бачимо всю інформацію в панелі Character.

З'ясування розмірів і відстаней - інструмент лінійка (Ruler Tool)

	1. Під час вимірів треба утримувати зажатим Shift.

Отримання даних про колір - інструмент піпетка (Eyedropper Tool)

	1. В налаштування зверху має бути:
		Sample Size: Point Sample
		Sample: All Layers
	2. Клікнувши в будь-якому місці, під панеллю інструментів бачимо колір точки кліку.

Експорт графіки, методика

	1. Копіюємо об'єкт в новий документ.
		Інструмент Move Tool і проставлена галочка зверху для Auto-Select
		Знаходимо необхідний слой або групу в панелі слоїв
		Щоб вибрати декілька слоїв, клікаємо по них, утримуючи CTRL
		Клікаємо правою кнопкою миші по назві слою та обираємо Duplicate Layer...
		У вікні:
			Document: New
			Name: пусте
		Створюється новий документ з розмірами макета, у якому буде тільки здубльований нами слой
	2. Підріжемо зайве.
		Меню Image - Trim - Transparent Pixels - OK
	3. Експортуємо для WEB.
		Меню File - Save for WEB (Ctrl+Shift+Alt+S)
		JPEG - для зображень, фото, де немає прозорості, можна більше стиснути
		PNG-24 - іконки, де необхідна прозорість, мають великі розміри

Хитрість швидкого зберігання дрібної графіки

	1. Покажемо панель Меню Window - Action (Alt+F9).
	2. Створюємо нову папку (маленька іконка внизу панелі) my actions.
	3. Обираємо будь-який слой.
	4. В панелі Action обираємо Create new action.
	5. Name: Export for WEB.
	6. Function Key: F2.
	7. Record.
	8. Правий клік по обраному слою - Duplicate Layer...
	9. У вікні - Document: New - Name: пусте.
	10. Меню Image - Trim - Transparent Pixels - OK.
	11. Меню File - Save for WEB (Ctrl+Shift+Alt+S).
		(Місце зберігання файлу та ім'я значення не має)
	12. Закриваємо файл без зберігання (NO).
	13. Зупиняємо Аction - Stop playing/Recording.
	14. Поруч з пунктом Експорт ставимо виводити діалог, щоб Фотошоп при експорті спитав нас про файловий формат та місце зберігання - Toggle dialog on/off.

Як користуватись?

	1. Обираємо будь-який слой.
	2. Натискаємо F2.
	3. Фотошоп сам його дублює в новий документ, підрізає і запускає інтерфейс експорту, де все прописуємо вже самі.
	4. Фотошоп сам закриє новий документ і ми повернемось до документу, з яким працюємо.

Текстура

	Спосіб 1
		1. Об'єднюємо всі слої - Меню Layer - Flatten Image.
		2. Збільшуємо масштаб.
		3. Інструмент Crop Tool - вибираємо фрагмент, що повторюється - Enter - далі можна експортувати.

	Спосіб 2
		1. Інструмент Move Tool і проставлена галочка зверху для Auto-Select.
		2. Знаходимо необхідний слой, у якому знаходиться текстура.
		3. Робимо по ньому подвійний клік.
		4. Бачимо у вікні текстуру, якою залито слой.
		5. Наводимо курсором на зображення текстури.
		6. У підказці бачимо розмір.
		7. Зберігаємо текстуру у слой Фотошоп - 'стрілочка' - New pattern... - додалась до переліку.
		8. Створюємо новий документ з розмірами тектури, що повторюється - Меню File - New - 102px x 102px - OK.
		9. Створюємо у ньому новий слой заливки тектури - Меню Layer - New Fill Layer - Pattern... - Name: будь-яке - ОК (текстура, відповідно, має бути та, що зберігали).

Швидке додавання стилів

	1. Вибираємо будь-який слой.
	2. Натискаємо праву кнопку мишки.
	3. Команда Copy CSS копіює стилі у буфер (але потім їх треба корегувати, бо є багато зайвого).
